# Программа Распознавания Лиц

Это приложение на Python использует нейросети (библиотека `face_recognition`) для обнаружения и распознавания лиц в реальном времени через веб-камеру.

## Возможности

- Отображение видеопотока с веб-камеры.
- Обнаружение лиц и выделение их квадратами.
- Распознавание сохраненных лиц и отображение их имен.
- Возможность добавить новое лицо в базу прямо из интерфейса.
- Сохранение базы лиц в файл для использования при следующем запуске.

## Требования

- Python 3.6+
- Веб-камера

## Установка

1.  **Клонируйте репозиторий или скачайте файлы.**

2.  **Установите зависимости:**

    Вам понадобятся библиотеки, указанные в `requirements.txt`.

    ```bash
    pip install -r requirements.txt
    ```

    **Важное примечание про `dlib` (зависимость `face_recognition`):**
    Установка `face_recognition` требует скомпилированной библиотеки `dlib`.
    -   **Windows:** Вам может потребоваться установить Visual Studio с C++ (Desktop development with C++).
    -   **Linux:** Установите cmake: `sudo apt-get install cmake`.
    -   **macOS:** `brew install cmake`.

## Запуск

Запустите файл `main.py`:

```bash
python main.py
```

## Как использовать

1.  После запуска откроется окно с видеопотоком.
2.  Если программа видит лицо, которое она не знает, над ним будет написано "Unknown" (Неизвестный).
3.  Чтобы сохранить человека:
    -   Убедитесь, что лицо видно и выделено рамкой.
    -   Введите имя в поле внизу окна.
    -   Нажмите кнопку **"Сохранить лицо"**.
4.  Программа запомнит лицо и при следующем появлении (даже после перезапуска программы) будет подписывать его введенным именем.

## Структура проекта

-   `main.py` - Основной файл запуска, содержит код интерфейса (GUI).
-   `face_system.py` - Логика работы с нейросетью и хранением данных.
-   `faces.pkl` - Файл базы данных (создается автоматически после первого сохранения).
